# Max Charts

Pure JavaScript candlestick chart library. Zero dependencies.

- Canvas-based rendering with HiDPI support
- Interactive zoom (scroll) and pan (drag)
- OHLC tooltip on hover
- Auto-resize or fixed size
- Works in browser and Electron

```
candlestick/           # Library
collections/           # Data files
instructions/          # Documentation
```

See [instructions/candlestick.md](instructions/candlestick.md) for usage.
